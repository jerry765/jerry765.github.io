    websocket 在 vue 中的使用

    发表于 2023-08-22 分类于 project 阅读次数： Disqus： 本文字数： 569 阅读时长 ≈ 2 分钟

    依赖

    socket.io-client

    使用

    1. 规范 socket api
    1. 设置 url
    1. 建立 socket 连接

    websocket 连接应在打开页面时建立，关闭页面时销毁，所以应选择在入口文件建立。

    1. 进入 socket 房间

    为保证浏览器性能，房间应该在进入特定路由时建立，离开特定路由时退出，所以选择在路由守卫文件编写相关逻辑。

    1. 监听数据

    后端通过 WebSocket 发送的数据在前端页面渲染的部分，应在前端对应页面监听。
