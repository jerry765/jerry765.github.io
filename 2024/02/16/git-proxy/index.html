Windows 设置 git 代理

查看代理地址和端口

  1. 打开控制面板
  2. 选择网络和Internet
  3. 选择Internet选项
  4. 弹出窗口中选择连接选项卡
  5. 连接选项卡中选择局域网设置
  6. 查看代理服务器设置（默认为本机地址7890端口）
